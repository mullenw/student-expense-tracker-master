Key Features:
- Add Expenses
Enter amount, category, and optional note
Automatically saves today’s date

- Date Filters
All — show all expenses
This Week — current calendar week
This Month — current calendar month

- Totals and Category Breakdown

Shows total spending for the selected filter
Shows spending grouped by category

- Edit and Delete
Tap Edit to update any expense
Tap X to delete an expense
All changes are saved using SQLite.

- How to Run:
run "npm install" on your computer's terminal

Then Start Expo:
"npx expo start"

Open Expo Go on your phone
Lastly, scan the QR code to load the app
